Как это работает

providedIn: 'root'
Единственный экземпляр на всё приложение. Все компоненты без локального провайдера получают его.
providers: [TokenService] на компоненте
Создаёт новый экземпляр для этого компонента и всех его потомков (Branch и Grandchild делят один экземпляр).
@Self
Angular смотрит ТОЛЬКО в инжектор текущего узла и не проверяет родителей вовсе. Если провайдера нет — ошибка (или null при @Optional).
@SkipSelf
Пропускает только собственный инжектор узла; дальше — обычный поиск вверх по родителям, и выигрывает первый найденный провайдер. Это не «взять у деда»: у Branch (свой провайдер есть) skipSelf даёт Root, а у Grandchild (своего провайдера нет) — всё равно экземпляр Branch, потому что поиск наверх идёт точно так же, как без модификатора.
@Optional
Если токен не зарегистрирован нигде в дереве — возвращает null вместо ошибки.
