Default ChangeDetectionStrategy.Default

Re-checked on every CD pass, regardless of input changes.

@for (node of defaultNodes(); track node.label) { }

OnPush ChangeDetectionStrategy.OnPush

Re-checked only when an input reference changes or a signal/event inside the component fires.

@for (node of onpushNodes(); track node.label) { }

Как читать счётчики

  • checked N times — сколько раз Angular вызвал шаблон компонента (dirty-check).
  • Local event — обновляет родительский signal; оба поддерева получают CD-проход, но OnPush-узлы проверяются только если их inputs изменились.
  • setTimeout — макро-задача тоже запускает CD-проход (zone.js или явный markForCheck).
  • Set signal input — меняет sharedInput, переданный как input() в оба поддерева; OnPush-узлы тоже перепроверяются.
