Легенда фаз

  • sync Синхронный код — выполняется сразу, пока стек не опустеет.
  • microtask Promise.then / queueMicrotask — сливаются все до следующей макротаски, в том числе между макротасками.
  • macrotask setTimeout / setInterval — берётся одна за проход цикла.
