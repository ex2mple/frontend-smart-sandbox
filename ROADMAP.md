# Roadmap

## Сделано

- **Движок песочниц**: создание песочницы из шаблона за секунду через companion dev-сервер + Vite HMR; поддержка мультистраничных песочниц (маршрутизация между несколькими страницами внутри одной песочницы); pin/unpin (перенос из расходной `generated/` в коммитящуюся `saved/`); удаление отдельной песочницы; полная очистка (`wipe`).
- **Дизайн-система**: единый набор CSS-токенов `--sb-*` (цвета, отступы, радиусы, шрифты) для всех песочниц, соответствие WCAG AA.
- **Dev-консоль**: встроенный оверлей, который в dev-режиме перехватывает `console.*` с указанием источника — не нужно открывать DevTools браузера, чтобы увидеть лог песочницы.
- **8 обучающих визуализаторов**, переведённых на replay-модель: event-loop, closures, prototype-chain, this-binding, signals, change-detection, lifecycle, di-tree. Каждый выполняет настоящий код, записывает реальный прогон и даёт пройти его пошагово через степпер с живой диаграммой; часть шаблонов дополнена карточками-предсказаниями, где вердикт вычисляется из фактического результата, а не подставляется заранее.

## Планы

- Новые темы визуализаторов — например, rendering-конвейер Angular или анимации переходов между страницами.
- Параллельная браузерная QA-инфраструктура — возможность гонять несколько независимых браузерных сессий одновременно вместо одной общей.
- Скриншоты и GIF в README, чтобы показать песочницы в действии без необходимости запускать проект локально.
